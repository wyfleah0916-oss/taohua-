## ✨ 它是什么

一个纯前端的互动娱乐网页应用。用户点击启动后，屏幕会涌入大量网络热梗、表情、代码片段和抽象符号组成的「缘分粒子流」，用户在合适的时机截屏捕获，系统根据捕获到的内容随机生成一份荒诞但莫名有道理的「量子桃花运报告」。

**关键词：** 赛博朋克风 · 随机生成 · 社交分享 · 纯前端 · 无需后端

## 🚀 如何运行

本项目是纯静态页面，无需安装任何依赖，无需构建。

**方式一：直接打开**

由于使用了 ESModule（`import/export`），需要通过 HTTP 服务访问，不能直接双击 `index.html` 打开。

```bash
# 用 Python 起一个本地服务
python3 -m http.server 8080

# 或者用 Node.js 的 serve
npx serve .
```

然后浏览器访问 `http://localhost:8080`。

**方式二：VS Code Live Server**

安装 Live Server 扩展 → 右键 `index.html` → Open with Live Server。
